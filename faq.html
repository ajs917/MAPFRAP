What tools have you used to build your website and how did you selected them?

BootStrap:
The template for this website was found on Start Bootstrap. The ‘Modern Business’ template was used. The template was utilised due to its appropriate configuration to employ cowboy coding principles for the required content of the website. For example, this section was well suited to the FAQ page and the Contact Us page had a validated form. Additionally, the template was free.

GitHub Desktop:
GitHub was used to host and publish the website. There are many benefits to using GitHub – its free, collaborative, implements version control and is simple to use for anyone. A repository was created that contains all the contents of the webpage, and GitHub desktop was used to push and pull commits as required when changes were made to the source code.

Atom Editor:
The editor used for website design was Atom Editor. Atom is a modern free text editor by GitHub. The software can be integrated wit GitHub to commit changes directly from the Atom interface – making it the ideal choice for editing.

Describe the steps to register and setup a domain name for your website

1) Select a domain name:
Key things to consider when selecting the domain name include:

  • Buy both the .co.uk and .com versions of your domain name to avoid confusion to users
  • Use keywords for your domain name, in this case the name of the product/company makes sense
  • Keep domain name short and memorable, avoid use of numbers and characters where possible

2) Check the domain name is available:

  • Check the domain name is not trademarked – this can be done on the Government Website. It may be worth cross checking across multiple domains to avoid legal trouble.
  • Check your domain name has not been taken. Nominet is the official registry of UK domain names and runs a service called WhoIs to check domain names.

3) Register the domain name using a registrar

  • Nominet keeps a list of official registrars. Anyone in the list can be used, and the process of registering varies by registrar.
  • Some registrars are accredited – it is recommended to use accredited registrars like GoDaddy, WebWiz, Google Domains etc.
  • Follow through the process of the registrar and purchase your required domain. Several top-level domains (org, co.uk, com, biz, net etc.) may be available, select as many as required.

4) Purchase the rights to the domain name

  • Select how long to keep the rights to the domain name
  • Pay a fee to the registrar to make the registration
  • Once the transaction is complete, you own the domain name

5) Link the domain to your website

  • Specify the IP address where users go to when using your domain. The process to do this varies depending on your registrar and host server (i.e GitHub Pages)
  • GitHub Pages has an option to set a custom domain under settings of the repository – input your register domain name in the Custom Domain field. This adds a CNAME file to source files
  • Configure your Domain Name System (DNS) by creating a CNAME record. This process varies depending on your registrar, but once this is done your domain should be properly configured
